Titolo
Git flow

Abstract
Quando ho iniziato ad utilizzare Git per il versionamento dei sorgenti non avevo bene idea di come sfruttarne le decantate potenzialita'  come la facilita' di branching o il commit locale finendo per usarlo come Svn. Grazie a Git-Flow ho finalmente capito cosa Git si aspettava da me! Git-Flow astrae un possibile paradigma d'uso per Git diminuendone immensamente la curva di apprendimento ed insegnando allo stesso tempo come si sfrutta Git.

EROGAZIONE
Il talk dimostra praticamente come utilizzare git-flow con esempi relativi a casi d'uso ricorrenti.
